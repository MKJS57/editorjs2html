### Examples
Before running the examples, ensure you have **Just** installed:  
```sh
cargo install just
```

- To run the **simple example**, use:  
  ```sh
  just run
  ```  
  or  
  ```sh
  cargo run --example simple
  ```  

- To run the **Actix example**, use:  
  ```sh
  just actix
  ```  
  or  
  ```sh
  cargo run --example actix
  ```
